**# StudyTrack AI – Student Habit Recommendation System**

Project Description
StudyTrack AI is a web-based application that tracks students' study habits and provides personalized study recommendations.

The system allows students to log in, view their study statistics, and receive recommendations based on their recorded data. An admin panel is available to monitor registered students.

--------------------------------------------------

Features

- Student Signup and Login
- Admin Login
- Study Log Tracking
- Study Hours Visualization using Plotly
- Personalized Recommendations
- Admin Dashboard to View Students

--------------------------------------------------

Tech Stack

Frontend
- Streamlit
- Plotly
- Pandas

Backend
- FastAPI
- Uvicorn

Machine Learning
- Clustering
- Recommendation Logic

--------------------------------------------------

Project Structure

MILESTONE_1 – Data Preprocessing
Milestone_2_Clustering – Student Segmentation
Milestone_3_Recommendation – Recommendation Logic
Milestone_4_WebInterface – Streamlit Application

--------------------------------------------------

How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run backend
uvicorn main:app --reload

3. Run frontend
streamlit run app.py
